Подключение
Мы даём возможность получать выдачу нового Яндекс Wordstat в виде сырых данных.

Начало работы

Для начала использования XMLRiver, выполните следующие действия:

Зарегистрируйтесь.
В аккаунте в разделе Настройки сбора настройте параметры сбора. Без указания этих параметров Вам будет недоступна ссылка для сбора данных.
Убедитесь, что запросы успешно отправляются:
Сформируйте и отправьте запрос методом GET с компьютера. Используйте URL для запросов, указанный в разделе Настройки сбора и добавьте к нему параметр query. Запрос будет выглядеть следующим образом:

http://xmlriver.com/wordstat/new/json?user=[user_id]&key=[key]&query=test
ВАЖНО! Строка, которая передается в параметр query не должна содержать амперсанд (&). Его нужно заменять на код %26
Проверьте полученный JSON-документ.
Ответ должен соответствовать заданному формату и не должен содержать ошибок.
Формат выдачи зависит от настроек поиска: топы запросов или динамика.
